## ie11 max-width

- ie11에서 max-width가 동작하지 않습니다.
- position: absolute; left: 0;으로 해결했습니다.

## ie11 text-align

- start와 end 값이 동작하지 않아서 left와 right를 사용합니다.

## ie11에서 web font 로딩되지 않음

- web font를 캐싱 하지 않을 경우 새로 고침 시에 나타나지 않아 캐시 하지 않는 것에서 이미지와 폰트를 제외 하였습니다.

## ie11 flex 축약형

### 일반적인 축양형

```css
# 아무것도 안쓰면;
flex: 0 1 auto;

# flex: 1;
flex: 1 1 0%;

# flex: auto;
flex: 1 1 auto;

# flex: initial;
flex: 0 1 auto
```

### ie11에서 축약형

```css
# 아무것도 안쓰면;
flex: 0 0 auto;

# flex: 1;
flex: 1 0 0px;

# flex: auto;
flex: 1 0 auto;

# flex: initial;
flex: 0 0 auto
```

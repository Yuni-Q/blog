## 0. html을 검사

- [http://validator.w3.org/](http://validator.w3.org/)

## 1. h1 ~ h6 태그 : 제목

- 크기와 상관없이 level로 사용해야 합니다.

## 2. 기본 태그

- a : 문서를 연결할 때 사용합니다.
- span : 인라인 영역으로 구분합니다.
- div : 박스 영역으로 구분합니다.
- p : 문단 정의합니다.

## 3. dl dt dd 태그 : 용어 설명용

- dl : 용어를 설명하는 목록입니다.
- dt: 용어의 제목입니다.
- dd: 용어 설명입니다.
- dt와 dd는 1:1, 1:n, n:1, n:m 가능합니다. 하지만 1:0은 불가능합니다.
  - dt에 대한 dd는 하나는 무조건 따라 와야 합니다.

## 4. ol ul li 태그 : 순차 목록

- ol : 순차 목록에 사용합니다.
- ul : 비순차 목록에 사용합니다.
- li : ol 또는 ul의 하위 항목으로 목록 요소입니다.

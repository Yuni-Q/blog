## function 키워드는 다양한 목적에 두루 쓸 수 있는 만능 키워드 입니다.

- ES5 이후로 기능을 좀 더 상세하게 나눠서 사용할 수 있습니다.
- 함수로 쓰고자 할 때는 arrow 함수
- 생성자로 쓰고자 할 때는 class
- 메서드를 쓰고자 할 때는 메서드 축약형
- function은 제네레이터 사용 시에만 사용(function\*)

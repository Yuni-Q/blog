## 환경 설정

### Homebrew 설치

```zsh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
```

### Node 설치

```zsh
brew install node
# node 버전 확인
node -v
# npm 버전 확인
npm -v
```

### expo-cli 설치

```zsh
npm install -g expo-cli
```

### 프로젝트 생성

```zsh
expo init app
```

### 안드로이드 스튜디오 설치

```zsh
brew cask install android-studio
```

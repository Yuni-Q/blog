- REST API를 사용할 때는 cURL 옵션 중 몇 가지를 꼭 알아두는 것이 좋습니다.
  - -i: 응답 헤더 출력 (옵션 없으면 응답 본문만 출력함)
  - -v: 중간 처리 과정, 오류 메시지, 요청 메시지와 응답 메시지를 헤더와 본문을 포함해 전체 출력
  - -X: 요청 메소드를 지정 (옵션 없으면 기본값은 GET)
  - -H: 요청 헤더를 지정
  - -d: 요청 본문을 지정 (옵션 없으면 요청 본문 없음)

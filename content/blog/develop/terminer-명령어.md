- pwd : 현재 위치
- cd (directory) : change directory
- mkdir (폴더명) : 폴더 생성
- touch (파일명) : 파일 생성
- sudo (명령어) : root 권한
- rm -rf (폴더명 or 파일명) : 모든 하위 폴더와 파일을 포함하여 삭제
- ls : 현재 디렉토리에 파일 및 폴더 표시
  - ls -a : 숨김 파일도 표시
  - ls -l : 파일 상세 표시
- cp : 파일 복사
- mv : 파일 이동

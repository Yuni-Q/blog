## [Flipper](https://fbflipper.com/)

- 페이스북이 만든 모바일앱 디버거도 좋음(다만 연결 시 에러나는 사람 다수 발견)
- setup doctor 문제 해결할 것
```zsh
npm i react-native-flipper redux-flipper rn-async-storage-flipper
npx pod-install # 아이폰 전용
```

- 핸들러 메소드 네이밍 : handle + 주체 + 이벤트 (ex: handleSaveButtonClick)
- 받아쓰는 컴포넌트 쪽에서는 on + 주체 + 이벤트 (ex: SaveButtonClick)
- 일반 함수(메서드) 네이밍 : 동사 시작 (ex: convertAToB)
